## 📊 ИСПОЛНИТЕЛЬНОЕ РЕЗЮМЕ

**uz++** — компилируемый язык программирования на узбекском языке, построенный на C++23.  
**Статус**: ✅ **ГОТОВ К ИСПОЛЬЗОВАНИЮ**

---

## 🏗️ АРХИТЕКТУРА И КОМПОНЕНТЫ

### Основные компоненты:
1. **Lexer** (`src/lexer.cpp`) - токенизация узбекского кода
2. **Parser** (`src/parser.cpp`) - построение AST (Abstract Syntax Tree)
3. **CodeGen** (`src/codegen.cpp`) - генерация C++23 кода
4. **Runtime** (`stdlib/apparat.hpp`) - стандартная библиотека
5. **LSP Server** (`src/lsp_server.cpp`) - поддержка VS Code
6. **DAP Server** (`src/dap_server.cpp`) - отладчик

### Поддерживаемые платформы:
- ✅ Windows
- ✅ Linux
- ✅ macOS
